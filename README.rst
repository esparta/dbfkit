DBF classes and utilities. For humans

* Free software: Apache license
* Documentation: http://dbfkit.readthedocs.org.

Features
--------

* Classes:
    * DBFUtils: Read a DBF and exposes as a csv file
* Console Scripts:
    * dbf2csv: Export a DBF file to csv (utf-8 encoding)
